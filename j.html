map方法是非常容易理解的，
		来看reduce方法和map方法的差别
		1. map方法的参数只有三个cur, index, arr 但是reduce方法多了一个prev, cur, index, arr
		2. map会对数组中每个数都执行一遍函数，reduce则是从第二个数开始执行

		其实记这些没有太大的用处，理解到reduce是一个合并的归一的方法才最重要。
		prev初始值就是第一个值，之后prev等于上一次执行函数的返回值。所以累加方法可以这样写：
		var sum = arr.reduce(function(prev, cur, index, arr) {
			return prev + cur
		})

		更精简一些 可以直接写
		arr.reduce((p, c) => p + c)

ES6模块语法：
			import ... from ...
			export ...

		默认方式:
		export default
		import default from '模块'

		一般方式：
		export 变量或函数声明
		import {变量或函数对应的名字} from '模块'
		需要注意的是，事实上这种到处方式应该也是导出了一个对象，然后导入的时候借助解构赋值去赋值

		两种方式可以混用
	
		另一点需要注意的是，npm安装的路径的node_modules目录下安装的东西，就是一个一个模块。
		通过上面的import语法可以直接使用这些模块提供的功能。这是使用别人智慧劳动的手段。
	
		需要注意的是，这里的模块导入并不会成功，因为存在跨域问题。
		因此，需要使用create-react-app 或者vue-cli 或者nginx服务器 或者用springboot。
	
		最后使用create-react-app做了一个模块的例子,放在react文件夹下，使用前请使用npm安装
		命令npm install

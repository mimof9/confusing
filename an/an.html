SVG是一种基于XML语法的图像格式，全称(可缩放矢量图)。
			本质上是文本文件，体积较小，放大不会失真。

			SVG 文件可以直接插入网页，成为 DOM 的一部分，然后用 JavaScript 和 CSS 进行操作。
		
			用法：
				1. SVG 代码都放在顶层标签svg之中
					viewBox属性的值有四个数字，分别是左上角的横坐标和纵坐标、视口的宽度和高度
				2. SVG 代码也可以写在一个独立文件中，然后用img、object、embed、iframe等标签插入网页。
				3. CSS也可以用SVG
				4. SVG文件还可以转换为BASE64编码，用作Data URI

			也就是说，SVG完全可以当作普通图片来使用
